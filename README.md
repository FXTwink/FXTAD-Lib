# 仓库说明
简介：本仓库旨在为团队提供统一、标准、经过验证的 Altium Designer 原理图库、PCB 封装库及 3D 模型，以提高 PCB 设计效率，减少因封装错误导致的打样返工。
本库标准仅满足内部使用，如需对外使用，请按对用标准核对修改。
# 📂 目录结构
```
.
├── 3D                  # 3D 模型文件
├── PcbLib              # PCB 封装库
├── SchLib              # 原理图符号库
├── FXT-DB.Dblib        # AD数据库对接文件
├── FXT-DB.xlsx         # 数据库xlsx版
└── README.md           # 本说明文档
```
# 🤝 贡献指南
如果你发现库中有错误，或者需要添加新元件，请按以下流程操作：
检查重复：在现有库中搜索，确认元件或封装不存在。
创建新元件：
复制同类元件作为模板进行修改。
严格遵守上述命名规范。
必须附带官方 Datasheet 链接或文件。
提交审核：
提交 Pull Request (PR) 或联系库管理员。
在 PR 描述中注明修改内容和 Datasheet 来源。
审核通过：管理员确认无误后合并至主库。
